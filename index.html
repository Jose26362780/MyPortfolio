Css

Como linguagem de marcação ele funciona para dar estilos as paginas

Javascript

É a stack em destaque que pode rodar do lado do navegador assim como o lado do servidor

React

Para a criação de interfaces agradaveis, bonitas e interesantes

Next.js

É um framework para React que oferece uma estrutura básica para projetos React.

Tailwind Css

É um framework CSS com classes predefinidas que você pode usar para construir e projetar páginas web diretamente na sua marcação.

Styled - component

É uma biblioteca poderosa para estilização de componentes em aplicações React.

Node.js

Para Rodar o Javascript do lado do servidor e construir aplicações robustas e escalaveis
